# demo-ai-record

A polished demo app for an AI Sales Visit Assistant.

## What it demonstrates

- Real-time note capture flow (mock recording/transcript)
- AI analysis output:
  - Polished summary
  - Structured insights
  - Smart action items
- Apply polished summary back into notes
- Save notes with toast feedback
- Create task actions
- Responsive UX for mobile, tablet, and desktop
- Desktop two-column workspace with internal content scrolling

## Tech stack

- Vue 3
- Vite
- Vant UI

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

This is a demo/prototype experience with mocked AI behavior and local persistence for presentation purposes.
